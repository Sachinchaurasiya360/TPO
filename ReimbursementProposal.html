1. Executive Summary

This proposal is submitted to request reimbursement for the operational and maintenance expenses incurred in developing and running the Vishwaniketan TPO Portal — a full-stack web application built to digitise and automate the entire Training & Placement process of the institute. The portal is live, production-deployed, and actively used by students, faculty, and placement administrators.

The charges listed herein cover only essential third-party infrastructure services required to keep the application running at the minimum viable production standard. No development labour costs are claimed. All rates reflect the lowest available commercial tiers of each platform.

2. Project Overview

The TPO Portal is a role-based, multi-user web platform that consolidates placement-related workflows under a single system accessible 24 × 7 from any device. It replaces manual paperwork, Excel sheets, and ad-hoc email communication with a structured, verifiable, and audit-friendly digital workflow.

Technology Stack

  • Frontend: React 19, TypeScript, Vite, Tailwind CSS, Framer Motion
  • Backend: Node.js, Express 5, TypeScript, Zod validation
  • Database ORM: Prisma with PostgreSQL
  • Authentication: JWT tokens with HTTP-only cookies, bcrypt password hashing
  • File Storage: Cloudinary (images, PDFs, resumes, marksheets)
  • Email Service: Resend (transactional email, OTP, notifications)
  • Deployment Platform: Vercel (frontend + serverless backend)
  • Database Hosting: Neon Serverless PostgreSQL
  • Domain: vimeettpo.xyz (.xyz TLD via domain registrar)

5. Justification for Each Service

5.1 Vercel Pro — ₹1,680/month

The Vercel free tier restricts serverless function execution time to 10 seconds and limits bandwidth to 100 GB/month. For an application with file uploads, database queries, and PDF generation, the Pro plan's 60-second function timeout and increased bandwidth allowance are the minimum viable requirements to prevent request timeouts and service disruptions.

5.2 Neon PostgreSQL — ₹350/month

The Neon free tier auto-suspends after 5 minutes of inactivity, causing cold-start delays on every first request. A minimal paid add-on keeps the database connection always-on and provides sufficient compute and storage for the application's active user base.

5.3 Cloudinary — ₹350/month

Every student account can store a profile photo, resume, semester marksheets (up to 8), certificate files, and achievement documents. At scale across hundreds of students, the free tier's 25 monthly credits are exhausted quickly. The paid plan provides the required storage, bandwidth, and transformation capacity to reliably serve files — particularly PDFs for marksheets and resumes — without upload failures or access restrictions.

5.4 Resend — ₹45/month

The system sends transactional emails for account approvals, password reset OTPs (security-critical), and job application status updates. A minimal paid tier is used purely to enable custom domain sending (required for SPF/DKIM email authentication) and to avoid the deliverability restrictions of the free tier.

5.5 Domain Registration (vimeettpo.xyz) — ₹900/year

A custom domain is essential for professional credibility, institute branding, and correct functioning of email authentication records (SPF, DKIM). Using a vercel.app subdomain as the permanent URL would undermine institutional identity and is inconsistent with the application being an official institute system.
